# كيف نفهم الإسلام — How We Understand Islam

Interactive trilingual web app based on **"Kayfa Nafham al-Islam"** (How We Understand Islam) by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- **20 Islamic concepts** organized by theme (basics, Ummah's problems, women, science, modernity)
- **"Who Wants to Be a Scholar?"** quiz with 15 questions and 3 lifelines
- **Two age modes**: Young Explorer (7-12) and Teen Scholar (13+)
- **XP and badges** gamification system
- **9 core principles** from the book
- **Book explorer** — interactive chapter map
- **Trilingual**: Arabic, English, French
- **3 themes**: Compass (ocean blue), Night, Dawn
- **Narrator**: audiobook mode with karaoke, voice settings, duo reading
- **Daily duas** and daily insight card
- **PWA-ready** with offline support

## Book Summary

The book presents Al-Ghazali's vision of Islam as a comprehensive, simple, and modern religion. Key themes include Islam's simplicity, diagnosing the Ummah's problems, Islam and science, critique of sectarian fanaticism, critique of harmful Sufism, women's rights, fighting despotism, social justice, and "Why I Am Muslim."

## Tech

- Vanilla HTML/CSS/JS — no frameworks
- Web Speech API for narrator
- Web Audio API for sound effects
- localStorage for persistence
- Responsive mobile-first design
